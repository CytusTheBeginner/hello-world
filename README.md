# This is my first project using VUE JS

## Cara Instalasi Vue JS

```
npm create vite@latest
npm install
npm run dev

untuk production : npm run build

```

## Fungsi File yang ada pada Folder Project

```
Package JSON berfungi untuk menyimpan library dan defedencies dari project kita.
Package JSON juga digunakan untuk konfigurasi dan running project.
Package Lock JSON berfungsi untuk cache dari file Package JSON. (jika dihapus maka akan dibuat lagi otomatis)
Index.HTML berfungsi sebagai container dari project/starting awal aplikasi.
Git Ignore berfungsi untuk mencegah file yang ada pada folder project supaya tidak dipush ke Repository

Src = folder untuk menyimpan file project
Src/Assets = untuk menyimpan file CSS dan Gambar
Src/Components = untuk file komponen kecil seperti header, footer, dan card
Src/App.vue = untuk file Starting aplikasi Vue
Src/main.js = sebagai pusat/inti untuk aplikasi yang kita kembangkan
Src/style.css = digunakan untuk styling
```
